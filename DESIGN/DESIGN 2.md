Here's a Project Brief (PRD Lite) for the VietNet Interior Website and Admin System based on our conversation:



Project Brief: VietNet Interior Website & Admin System (Phase 1)

Project Title: VietNet Interior Website & Admin System

Date: October 26, 2023

Product Manager: [Your Name/Role]



1. Project Overview

This project aims to develop a sophisticated and user-friendly website for Nội Thất VietNet, featuring a premium design for end-users (Viewer Mode) and a robust content management system for administrators (Edit Mode & Admin Dashboard). The goal is to enhance brand presence, streamline content updates, and efficiently manage customer consultations.

2. Project Goals & Objectives





To provide a high-end, intuitive website experience for VietNet customers.



To empower VietNet administrators with easy-to-use tools for content editing and lead management.



To maintain a consistent, luxurious brand identity across all user interfaces.



To improve efficiency in tracking and responding to customer consultation requests.

3. Target Audience





Customers (Viewer Mode): Individuals and businesses seeking interior design and construction services, looking for a professional and visually appealing online presence.



VietNet Administrators/Staff (Edit Mode & Admin Dashboard): Internal team responsible for website content updates, lead management, and overall site administration.

4. Core Features & Modules (Phase 1)

The following UI components and functionalities have been designed and confirmed:

4.1. VietNet Homepage - Viewer Mode (Customer Experience)





Description: The public-facing website interface designed for a clean, premium, and non-technical user experience.



Key Components:





Header: Consistent, elegant header with "VietNet Interior" logo (serif font) and sticky navigation (TRANG CHỦ, VỀ CHÚNG TÔI, THIẾT KẾ NỘI THẤT, NGÔN NGỮ THIẾT KẾ, THI CÔNG NỘI THẤT, TỦ BẾP) with subtle icons.



Hero Section: Prominent, high-quality living room image with a clear Call-to-Action (CTA) for "Nhận Tư Vấn" (Get Consultation).



Content Sections: Dedicated sections for "About Us" (company history), "Featured Projects" (grid with hover effects), and a combined "Contact & Free Consultation" section (form, office info, map).



Floating Contact Widget: Positioned at the bottom-right, featuring Zalo, Facebook, and Call icons with a subtle vibrating animation effect to attract user interaction without disruption.



Design Principle: Minimalist, luxurious aesthetic with a primary Brown and White color theme. Focus on crisp imagery and perfectly aligned content blocks.

4.2. VietNet Homepage - Edit Mode (Admin Content Management)





Description: The administrative interface for direct, inline content editing on the homepage.



Key Components:





Admin Toolbar: A brown toolbar at the top with "LƯU THAY ĐỔI" (Save Changes) and "HỦY" (Cancel) buttons.



Inline Editing Indicators: All editable content regions (images, titles, text blocks) are visually highlighted with light brown dotted borders and a small "pencil" icon at the top-right corner.



Editable Elements: Allows direct modification of images, titles, and article content.



Floating Contact Widget: Remains visible, with an "edit settings" icon alongside it for configuration.



Design Principle: Clear visual cues for editable areas, intuitive saving/canceling mechanisms, maintaining the brown-white professional theme.

4.3. Admin Dashboard - Overview





Description: The central hub for VietNet administrators, providing a quick summary of key website metrics and management options.



Key Components:





Sidebar Navigation: Includes icons and links for: "Tổng Quan" (Overview), "Quản Lý Bài Viết" (Post Management), "Quản Lý Tài Khoản" (Account Management), "Thống Kê Truy Cập" (Traffic Stats), "Danh Sách Tư Vấn" (Consultation List), "Cài Đặt Menu" (Menu Settings).



Main Content Area: Displays a summary of daily statistics: Total Visitors, New Consultations (with a list of recent names/phones), and Post Status.



Design Principle: Clean, organized layout with a white background and brown accents for headers and buttons, consistent with the brand's professional identity.

4.4. Admin Dashboard - Consultation Management





Description: A detailed section within the Admin Dashboard specifically for managing customer consultation requests.



Key Components:





Consultation Table: Lists all potential leads with columns for: Name, Phone, Service Interest, Date, and Status (New, Contacted, Completed).



Action Buttons: For each row, provides quick actions: "Call Now", "Send Email", "Delete".



Search & Filter: A search bar and a date filter option at the top for easy lead navigation.



Design Principle: Highly functional, clear, and organized UI to facilitate efficient lead tracking and follow-up, adhering to the established brown-white color palette.

5. Brand Identity & Design Principles





Color Palette: Predominantly Brown and White, conveying luxury, warmth, and professionalism.



Aesthetics: Clean, minimalist, high-end, and sophisticated.



User Experience: Intuitive navigation, clear calls-to-action, seamless content editing.



Responsiveness: (Implicit) Designs should ideally be adaptable for various screen sizes.

6. Future Considerations (Potential Phase 2 Features)





Product Category Page (e.g., Tủ Bếp): Design a detailed product listing page with filtering options (e.g., by material).



Dashboard Enhancements: Integration of traffic growth charts/analytics into the Overview dashboard.



Admin Role Management: Design an interface for managing user roles and permissions within the Admin Dashboard.



Theme Picker for Admin: Allow administrators to change the website's color scheme.



Detailed "Thi Công Nội Thất" Page: Design a specific page outlining a 5-step process for interior construction.



This brief provides a clear understanding of the scope and requirements for the initial phase of the VietNet Interior Website and Admin System.
