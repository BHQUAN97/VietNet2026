# 🛒 TỔNG QUAN DỰ ÁN: NỘI THẤT VIETNET
> Tham chiếu phong cách: nlhome.vn | SEO-Focus | High-End UI

## 1. MỤC TIÊU HỆ THỐNG
- **Public Web:** Giới thiệu công trình, sản phẩm nội thất. Tối ưu SEO điểm tuyệt đối (90+ Mobile).
- **Admin Panel:** Quản lý toàn bộ nội dung, tùy chỉnh trang chủ linh hoạt (Page Builder), theo dõi traffic.
- **Quy mô:** 300 users/ngày, 5 Admin. Team 1 Dev + Claude AI.

## 2. STACK CÔNG NGHỆ (TECH STACK)
- **Frontend:** Next.js 14/15 (App Router), Tailwind CSS, Shadcn UI, Framer Motion.
- **Backend:** NestJS (Node.js), BullMQ (Xử lý hàng đợi).
- **Database:** MySQL 8.0 (Lưu trữ chính), Redis (Caching, Socket, Counter).
- **Email:** Resend API (Gửi mail tư vấn).
- **Real-time:** Socket.io (Thông báo Admin).
- **Infrastructure:** Docker Compose, Nginx (Micro-caching 1s), Cloudflare CDN.
- **Hardware:** 4 Core CPU, 16GB RAM, 160GB SSD.
